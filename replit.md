# Kirana Store Project

A premium Indian kirana grocery store website showcasing products with a modern, high-quality design.

## Features

-   **Product Catalog**: Browse rice, spices, dry fruits, and snacks.
-   **Categories**: Organized navigation for easy browsing.
-   **Premium Visuals**: High-quality imagery with soft shadows and minimal backgrounds.
-   **Responsive Design**: Works on mobile and desktop.

## Tech Stack

-   **Frontend**: React, Tailwind CSS, Framer Motion, Shadcn UI
-   **Backend**: Node.js, Express, Drizzle ORM
-   **Database**: PostgreSQL
-   **Images**: AI-generated premium product photography

## Setup

1.  Database is provisioned and seeded automatically.
2.  Images are located in `client/public/images`.
3.  Run `npm run dev` to start.
